# 小游戏框架

## 定位

小游戏是开放世界中的互动强化方式，不是考试关卡。小游戏帮助孩子反复接触单词、字母、语音和场景记忆。

## 插拔式架构

所有小游戏应实现统一接口：

```text
start_minigame(config)
complete_minigame(result)
return_to_scene()
write_learning_result()
give_reward()
```

小游戏必须能接入：

- Quest。
- A-Z anchor。
- Memory Card。
- Pet/Home/Shop reward。
- Learning Record。
- Parent/Admin summary。

## 字母贪吃蛇示例

目标：

- 吃到正确字母。
- 吃到正确单词。
- 避开干扰项。
- 分数越高，奖励越多。

结果：

- 小分数：获得基础 coins 或卡片 spark。
- 中分数：收集 memory card。
- 高分数：获得 shiny card、更多 coins 或特殊装饰。

孩子端没有失败，只是奖励不同。
