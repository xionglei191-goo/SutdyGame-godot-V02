# 美术风格与换肤预留

## 初期风格

V02 初期只采用一种美术风格，避免范围失控。

目标用户是 10 岁左右、四年级小女孩。风格关键词：

- 可爱。
- 温暖。
- 明亮。
- 梦幻但不幼稚。
- 小镇生活。
- 宠物陪伴。
- 收集感。
- 轻 2.5D。
- 绘本感。

## 统一范围

同一阶段内需要统一：

- 地图。
- 角色。
- 宠物。
- NPC。
- UI。
- 图标。
- 记忆卡。
- 小游戏。
- 音频情绪。

不同时尝试像素风、3D Q 版、扁平卡通、水彩等多套方向。

## 换肤预留

虽然初期只实现一种风格，但资产系统不能把具体路径写死在玩法逻辑里。

建议设计：

```text
ThemeProfile
AssetResolver
```

运行时通过接口取资源：

```text
get_npc_sprite("mina")
get_pet_sprite("sunny")
get_tile_atlas("world")
get_card_art("a_apple_core")
get_ui_skin("primary_button")
```

未来可以替换主题：

- storybook。
- clay。
- anime-lite。
- pixel。
- 3d-chibi。

但 V02 第一阶段只实现一个主题。
