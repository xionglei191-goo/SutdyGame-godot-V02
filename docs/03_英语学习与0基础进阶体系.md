# 英语学习与 0 基础进阶体系

## 总原则

英语学习应该循序渐进，但孩子端不展示强难度和强考核。系统内部可以记录熟练度、复现频率和语言复杂度，孩子端只感受到奖励、世界变化、宠物成长、卡片收集和故事推进。

## 0 基础假设

游戏默认孩子可能：

- 不完整认识 A-Z。
- 不会自然拼读。
- 不会独立读单词。
- 不会完整英文句子。
- 听英文时需要大量图片和动作支持。

因此 V02 需要补齐教材前置内容，而不是直接从三年级下开始。

## 学习阶梯

内部学习阶梯建议为：

1. 声音环境适应：听到英语，建立不排斥感。
2. A-Z 字母识别：字母形状、声音、核心图像锚点。
3. 字母 + 图片 + 单词：A = Apple 这类稳定编码。
4. 高频生活词：家庭、宠物、食物、玩具、颜色、数字、动作。
5. 简单行动句：Feed the pet、Open the box、Take the bus。
6. 场景对话：NPC 帮忙、商店购买、交通选择、家庭互动。
7. 教材预习与巩固：围绕现有素材进行生活化任务设计。
8. AI 互动与表达：后期在安全边界内做开放对话和跟读。

## 学习状态模型

系统内部可以记录：

- Seen：看见过。
- Heard：听过。
- Touched：点过或互动过。
- Played：在小游戏中用过。
- Remembered：复现过。
- Collected：记忆卡正式收集。
- Shiny：熟练后获得强化卡或更高奖励。

孩子端不显示这些术语。孩子看到的是卡片点亮、宠物开心、道具增加、故事记忆保存。

## 教材映射方式

现有教材素材作为中期目标和预习/巩固来源。内容进入游戏的路径为：

```text
curriculum item
  -> world place
  -> quest event
  -> dialogue / voice
  -> minigame review
  -> memory card
  -> parent/backend record
```

教材内容不直接以课本形式出现，而是转化为生活事件。
