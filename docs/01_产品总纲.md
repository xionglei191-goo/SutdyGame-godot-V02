# StudyGame V02 产品总纲

## 一句话定位

StudyGame V02 是一款面向儿童的移动端小镇生活 RPG / 养成游戏，体验目标接近“动物森友会式”的温暖、低压力、可长期停留的生活感。

玩家在 Sunshine Town 中拥有自己的家，结识动物居民，照顾伙伴，收集资源，完成轻委托，购买家具和服装，布置房间与小镇。英语内容作为生活环境自然出现，而不是主线考核目标。

## 核心判断

V02 的第一目标不是“把英语学习闭环跑通”，而是让孩子愿意每天回到一个可爱的生活世界。

英语学习必须服务于生活体验：招牌、物品名、NPC 短对白、收藏图鉴、小游戏和语音都可以出现英文，但不能把孩子推入背单词、闯关、考试或课程表。A-Z 记忆宫殿是世界地图的底层记忆编码，每个字母必须有稳定锚点，让孩子在探索、路过、装饰、收集和故事中潜意识建立路径记忆。

## 产品原则

- 生活养成优先，学习隐形融入。
- 玩家先是小镇居民，不是学生。
- 每天有轻量目标，但不强制通关。
- 家园、装饰、收集、NPC 关系和宠物陪伴是长期驱动力。
- 英语只出现在自然语境：物品、动作、问候、商店、委托、图鉴和可选小游戏。
- A-Z 记忆宫殿是地图骨架和潜意识学习方法；每个字母有对应世界锚点，但不以考试或背诵形式显性推进。
- 每个新单词都必须挂回对应字母锚点，用地图地点、角色事件和故事画面讲解，不能孤立加词。
- Memory Card 是收藏图鉴和故事回忆，用来承接锚点体验。
- Letter Snake 降级为可选小游戏，不作为购买食物或推进主线的唯一来源。
- 家长后台不进入孩子主流程。
- 移动端手机和平板优先，Godot 开发，触屏体验优先。
- 暂时本地游戏，预留账号、联网、AI 和云端扩展空间。

## 产品不是

- 不是考试系统。
- 不是课程表。
- 不是背单词列表。
- 不是学校 App。
- 不是强制通关游戏。
- 不是开放陌生人社交平台。
- 不是以小游戏刷分为主的学习产品。

## 核心系统

- Player Life System：角色移动、背包、每日状态、家园归属。
- Town System：Home、Town Plaza、Shop、Forest、Beach 等可回访区域。
- Memory Palace Map：A-Z 世界锚点、路线记忆、地标物件和可回访场景编码。
- Home Decoration：家具、摆放、房间主题、收藏展示。
- NPC Relationship：动物居民、固定对白、好感、送礼、生日/节日、轻委托。
- Resource & Crafting：采集、钓鱼/捡拾占位、材料、家具制作预留。
- Pet/Companion：宠物需求、陪伴反馈、外观和互动。
- Shop & Economy：金币、商品轮换、家具/服装/食物购买。
- Collection Album：物品、记忆卡、A-Z 彩蛋和照片式收藏。
- English Environment：自然英文物品名、短句、语音和可选小游戏。
- Parent/Admin Layer：设置和本地摘要，不进入孩子主流程。

## 当前保留与降级

保留：Godot 工程、SaveService、地图 JSON、AssetResolver、NPC profile/memory、VoiceService、Pet/Shop/Coins、Memory Album、测试框架。

调整：A-Z 记忆宫殿不再表现为显性学习主线，但必须作为世界地图的稳定底层结构保留；Memory Card 承接为收藏图鉴和故事回忆；Letter Snake 和学习记录只作为可选活动或后台记录。

后续开发以“小镇生活 MVP”为准：角色可移动、可探索、可与 NPC 互动、可收集资源、可购买/摆放家具、可保存家园状态，同时确保 A-Z 锚点在地图上有固定位置和可感知的场景表现。
