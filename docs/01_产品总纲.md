# StudyGame V02 产品总纲

## 一句话定位

StudyGame V02 是一款面向 0 基础儿童的移动端英语环境型生活冒险游戏。

它以 Sunshine Town 和 A-Z 记忆宫殿为世界骨架，通过家、宠物、小镇、朋友、交通、小游戏、语音环境和 AI NPC，让孩子在开放式生活体验中自然接触、理解、复现和使用英语。

## 核心判断

V02 不是强考核学习 App，不是单词闯关软件，也不是学校导览游戏。

V02 的第一目标是让孩子愿意长期待在一个英语生活世界里。兴趣是第一导师，学习通过体验、重复、奖励、收集、故事和环境自然发生。

## 产品原则

- 兴趣优先，考核隐形。
- 从 0 基础开始设计，不假设孩子已经掌握字母、自然拼读或课本前置内容。
- A-Z 记忆宫殿是固定认知编码底座，从第一天植入。
- 其他玩法保持开放，孩子可以长期停留在基础区域，也可以逐步探索更远区域。
- 英语学习藏在生活行动、语音环境、NPC 互动、小游戏、记忆卡和奖励循环里。
- 家长后台不进入孩子主流程。
- 移动端手机和平板优先，Godot 开发，触屏体验优先。
- 暂时自用，不考虑商业模式。
- 暂时本地游戏，预留账号、联网、AI 和云端扩展空间。

## 产品不是

- 不是考试系统。
- 不是课程表。
- 不是背单词列表。
- 不是学校 App。
- 不是强制通关游戏。
- 不是开放陌生人社交平台。

## 核心系统

- World System：Home、Sunshine Town、School、Transport、World。
- A-Z Memory Palace：固定字母编码、世界锚点、故事记忆。
- Memory Card Collection：单词卡、画面、故事、语音、小游戏记录。
- English Environment：TTS、录音、后期跟读 AI、语音素材。
- NPC System：固定对白、AI 预留、NPC 记忆文件。
- Minigame System：可插拔小游戏，例如字母贪吃蛇。
- Pet/Home/Shop Loop：宠物需求、金币、道具、装饰、收集。
- Parent/Admin Layer：设置页或 NPC 激活后的家长后台。
- Data/Editor Tooling：Godot 内地图编辑，数据同步，运行时生成。
