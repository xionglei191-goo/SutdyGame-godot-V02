# 家长后台与本地数据

## 总原则

家长后台不进入孩子主游戏流程。孩子端只看到奖励、宠物、小镇变化、卡片、NPC 和故事。家长端通过设置页或特定 NPC 激活后进入后台。

## 家长入口

可选方式：

- 设置页。
- 家长 NPC。
- 长按入口。
- 简单家长验证。

进入后显示 Parent/Admin Dashboard。

## 家长可见内容

- 游戏时间。
- 接触过的词汇和句型。
- 记忆卡收集状态。
- 小游戏表现摘要。
- 语音练习记录。
- AI NPC 对话摘要。
- 宠物和任务进展。
- 建议复现内容。

## 本地优先

V02 初期是本地游戏，不做真实账号和联网后台。

但数据结构预留：

- `player_id`
- `profile_id`
- `save_slot`
- `learning_record_id`
- `account_adapter`

未来可以替换成本地/云端实现：

```text
SaveServiceLocal -> SaveServiceCloud
ParentDashboardLocal -> ParentDashboardBackend
NPCMemoryLocal -> NPCMemoryCloud
```

## 隐私原则

即使暂时自用，也需要从架构上尊重儿童数据边界：

- 录音可开关。
- AI 可开关。
- 联网可开关。
- 家长可查看摘要。
- 未来支持删除数据。
