# 内容基线整理与首批内容规划

> 状态：旧学习内容基线归档。  
> 当前方向已调整为生活 RPG / 小镇养成。本文中的 A-Z、Memory Card 和 Letter Snake 内容可作为收藏、图鉴、环境英文和可选小游戏素材，但不再决定主循环优先级。新的首批内容应围绕 Home、Town Plaza、Shop、动物居民、资源、家具和家园装饰重建。
> 后续新增任何单词内容时，必须补齐 `letter`、`core_anchor_id`、`world_place_id`、`story_memory`、`visual_hook` 和 `review_path`，确保故事围绕记忆宫殿编码展开。

> 项目：StudyGame V02  
> 目的：把现有 curriculum 和 A-Z 记忆宫殿策划转化为 V02 第一阶段可执行内容基线。  
> 范围：只规划首批内容，不生成完整教材库，不直接进入 Godot 实现。

## 1. 内容来源

V02 第一阶段只使用两个核心来源：

```text
curriculum/小学英语重点分析/
docs/assets/Sunshine_Town_And_The_World_A-Z字母记忆宫殿策划_v0.1.md
```

其中：

- A-Z 策划作为固定记忆编码基线。
- 三年级下册作为第一批教材预习/巩固主要来源。
- 四年级上册作为 Home、School、Food、Animal、Pet、Library 等主题扩展参考。
- 其他年级资料暂时只作为远期内容储备。

## 2. 内容原则

- 孩子端不出现 lesson、unit、word list、test、review 等强学习表达。
- 所有英语内容必须包装成生活事件、语音环境、NPC 对话、小游戏、记忆卡或道具反馈。
- 首批内容从 0 基础开始，不假设孩子已经掌握三年级前置内容。
- A-Z core anchor 从一开始植入，但不要求孩子一次掌握完整 26 个。
- 第一阶段内容服务于 Home / Pet / Town / Shop / Letter Snake 主循环。

### 10.1 Letter Snake 第一批目标

| 组别 | Target Letters | Target Words | 目的 |
|---|---|---|---|
| Home Set | A, C, D | apple, clock, dog | Welcome/Home/Pet |
| Food Set | A, O, E | apple, orange, egg | Supermarket |

### 10.2 奖励规则

- 任意分数都有 coins。
- 命中目标词增加 card spark。
- 高分可直接 collected。
- 连续多次高表现可获得 shiny card。
- 不显示 failed / wrong / test。

## 12. 第一阶段内容范围

必须进入第一阶段：

- 8-9 个首批 A-Z core anchor 互动。
- 约 20 张首批 Memory Card。
- Home / Pet / Supermarket / Pet Shop / Bookshop / Bus Station。
- 宠物食物循环。
- Letter Snake 第一版。
- TTS/录音/AI 只做数据和 stub。

- 完整 26 个 anchor 互动。
- 多小游戏。
- 真实 TTS、录音和跟读。
- 真实 AI NPC。

## 14. 验收标准

内容基线通过标准：

- 每个首批事件都能追溯到 A-Z、0 基础补齐或 curriculum 来源。
- 首批卡片数量控制在约 20 张，不扩大到完整教材库。
- 首批场景围绕 Home/Pet/Town，不把 School 变成产品脸面。
- 首批小游戏只做 Letter Snake。
- 孩子端表达保持生活冒险语言，不出现 lesson/test/word list。
- 家长和系统内部可以记录学习状态，但孩子端只看到奖励和收集。
