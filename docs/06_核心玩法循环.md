# 核心玩法循环

## 总原则

V02 的核心是生活 RPG / 小镇养成。孩子打开游戏后，首先感受到的是“我住在这个小镇里”，而不是“我要完成学习任务”。

## 主循环

```text
进入小镇
  -> 移动探索
  -> 和动物居民互动 / 接轻委托
  -> 收集资源或购买物品
  -> 获得 coins、材料、家具、服装、收藏
  -> 装饰家园 / 照顾宠物 / 提升 NPC 关系
  -> 解锁新的对白、区域、商品和小镇变化
  -> 第二天继续生活
```

## 首个 MVP 循环

```text
起床在 Home
  -> 出门到 Town Plaza
  -> 和 Mina 打招呼
  -> 捡到 3 个材料
  -> 去 Shop 买一件家具
  -> 回家摆放家具
  -> 保存房间变化和 Mina 好感
```

这个循环必须先跑通，再扩展小游戏、AI、家长后台或复杂地图编辑器。

## 奖励设计

奖励服务生活感：

- coins。
- materials。
- furniture。
- outfit。
- room decor。
- NPC relationship。
- pet mood。
- collection album。
- town unlocks。
- story memories。

低表现或慢节奏也能获得进展。没有失败惩罚，没有学习压力标签。

## 英语融入方式

英语只作为自然环境出现：

- 物品名：apple、chair、bed。
- 动作短句：Good morning、Thank you、Let's go。
- 商店和委托中的简单短语。
- 收藏图鉴里的英文标签。
- 可选小游戏中的轻量英文元素。

孩子端不显示 hard / easy / lesson / test / failed / wrong 等压力表达。
