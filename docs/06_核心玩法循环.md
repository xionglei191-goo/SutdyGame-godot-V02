# 核心玩法循环

## 总原则

V02 是重体验、轻考核的开放式游戏。学习通过生活需求、自然任务、小游戏、奖励、收集和世界变化发生。

## 基础循环

```text
生活需求
  -> 自然任务
  -> 小游戏互动
  -> 奖励
  -> 宠物 / 家 / 商店 / 卡片反馈
  -> 自愿继续玩
```

## 宠物示例循环

```text
领养宠物
  -> 宠物饿了
  -> 需要购买食物
  -> 需要 coins
  -> 做任务或玩字母贪吃蛇
  -> 单词越熟，积分越高
  -> 获得更多奖励
  -> 买更多食物、玩具、装饰
  -> 宠物更开心，家更丰富
```

孩子可以永远在新手村，也可以逐步挑战更远区域。系统不强迫升级。

## 奖励设计

奖励承载学习反馈：

- coins。
- pet items。
- room decor。
- outfit。
- memory cards。
- shiny cards。
- NPC relationship。
- new routes。
- story memories。

低表现也给小奖励，高表现给更多奖励。不使用失败惩罚作为主反馈。

## 开放式成长

孩子端不显示 hard / easy / level 这类压力标签。系统内部可以有隐形复杂度：

- 单词熟悉度。
- 句子长度。
- 小游戏节奏。
- 奖励倍率。
- NPC 英语比例。
- 语音重复次数。
