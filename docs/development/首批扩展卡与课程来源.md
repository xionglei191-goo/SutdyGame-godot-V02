# 首批扩展卡与课程来源

> 任务：V02-AZ-002  
> 数据文件：`data/cards/first_batch_extension_cards.json`  
> 范围：只补齐第一阶段主循环和 Letter Snake 已引用的非 core Memory Card。

## 数据原则

- A-Z core card 继续由 `data/cards/az_core_cards.json` 维护，固定 A-Z 编码不在本文件改动。
- 扩展卡用于 Home / Pet / Supermarket / Town / Weather 等生活事件，不要求绑定固定 A-Z anchor。
- 每张扩展卡必须包含 `word`、`story_memory`、`collection_state`、来源字段，以及 `minigame_refs` / `quest_refs`。
- 有明确教材来源时使用 `source_curriculum_refs`；0 基础桥接但暂未绑定教材单元时使用 `source_fill_needed`。

## 引用覆盖

当前首轮内容中，非 core card 引用来自：

- `data/minigames/letter_snake_config.json`
- `data/quests/pet_food_loop.json`

`tests/test_content_references.gd` 会把这些引用解析为 card_id，并要求它们全部存在于 A-Z core cards 或本扩展卡文件中。`tests/test_first_batch_extension_cards.gd` 负责扩展卡自身 schema、来源、初始收集状态、ID 唯一性和引用覆盖检查。
