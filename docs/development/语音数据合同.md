# 语音数据合同

> 范围：Round 8 `V02-VOICE-001/002`。第一阶段只提供本地数据和 mock 服务，不接真实 TTS、录音、发音测评、权限申请或网络。

## 数据文件

运行时语音数据位于：

```text
data/voice/voice_lines.json
```

根对象字段：

| 字段 | 类型 | 说明 |
|---|---|---|
| `schema_version` | number | 当前为 `1`。 |
| `voice_lines` | array | VoiceLine 列表，`audio_id` 唯一。 |

VoiceLine 字段：

| 字段 | 类型 | 说明 |
|---|---|---|
| `audio_id` | string | 稳定语音 ID，也是服务查询主键。 |
| `text` | string | 孩子端可播放的英文文本。 |
| `tts_voice_id` | string | 预留 TTS 声线 ID；当前仅用于 mock 返回。 |
| `speaker_id` | string | 说话者 ID，后续与 NPC profile 对接。 |
| `recording_enabled` | bool | 该句是否允许进入录音流程；当前仍不申请权限。 |
| `repeat_after_me_enabled` | bool | 是否适合跟读入口；当前仅做数据预留。 |
| `speech_eval_enabled` | bool | 是否允许发音测评；第一阶段固定为 `false`。 |

首批语音共 10 条：

```text
Good morning.
How are you?
Sunny is hungry.
Let's feed Sunny.
Here you are.
Thank you.
Let's go.
I go by bus.
Find apple.
You found a memory card.
```

## VoiceService Stub

脚本：

```text
scripts/systems/voice_service.gd
```

主要接口：

| 方法 | 返回 |
|---|---|
| `is_loaded()` | 数据是否加载且校验无错误。 |
| `list_voice_lines()` | VoiceLine 列表。 |
| `get_voice_line(audio_id)` | 按 `audio_id` 返回 VoiceLine；未知 ID 返回 `{ "ok": false }`。 |
| `mock_tts(audio_id)` | 返回本地 TTS 占位结果，`network_used=false`、`permission_requested=false`。 |
| `mock_play(audio_id)` | 基于 `mock_tts` 返回播放占位结果，不创建真实 `AudioStream`。 |
| `mock_recording(audio_id, duration_seconds=0.0)` | 对 `recording_enabled=true` 的语音返回录音占位结果；禁用录音的语音返回 `recording_disabled`。 |

第一阶段约束：

- 不申请麦克风权限。
- 不联网。
- 不保存真实录音文件。
- 不做真实 speech evaluation。
- 不硬编码具体音频资产路径。
