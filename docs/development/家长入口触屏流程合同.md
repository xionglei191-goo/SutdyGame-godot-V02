# 家长入口触屏流程合同

> 范围：`V02-PARENT-001` / `V02-PARENT-003`。本合同只定义入口边界和孩子端可见性；完整家长 dashboard UI 归 `V02-PARENT-004`。

## 入口原则

- 家长后台不得出现在孩子端主导航、生活面板、地图交互、可选活动或宠物循环中。
- 孩子端默认只显示生活 RPG、小镇探索、NPC、收集、商店、家园和可选活动。
- 家长入口使用本地 gate，不需要账号、网络、录音或真实 AI。
- 家长入口只读取本地 `ParentDashboardStore` 摘要数据；真实 dashboard UI 未实现前，不在主场景打开报告界面。

## 触屏流程

当前规格由 `Main.get_parent_entry_spec()` 返回：

- `entry_id`: `local_parent_gate`
- `trigger`: `reserved_settings_long_press`
- `hold_seconds`: `3`
- `confirmation`: `local_adult_confirmation_stub`
- `opens`: `parent_dashboard_local_summary`
- `child_flow_visible`: `false`
- `available_in_child_nav`: `false`
- `network_required`: `false`
- `account_required`: `false`

## 孩子端禁显词

孩子端主流程不得出现以下可见入口或报告文案：

- `Parent`
- `Dashboard`
- `Report`
- `后台`
- `家长`
- `报告`

例外：开发文档、测试名、服务类名和本地数据字段可以包含这些词；禁显范围只针对主场景孩子端可见 UI 文本。

## 验收

- 主场景不创建 `ParentButton`。
- 主场景可见 UI 文本不包含禁显词。
- `get_parent_entry_spec()` 明确 `child_flow_visible == false` 且 `available_in_child_nav == false`。
- `ParentDashboardStore` 仍可通过服务级测试读取本地摘要。
