# 家长后台本地摘要合同

`ParentDashboardStore` 是 V02 初期的本地数据读取 stub，只从 `SaveService` 的 `profile`、`game_state` 和 `learning_record` 生成家长端摘要字典。

## 边界

- 不进入孩子主流程，不创建场景、不挂接儿童端 UI。
- 不启用真实账号、联网、录音或云端后台。
- 不保存新的学习数据；只读取本地存档并归一化为摘要。

## 输出结构

```text
{
  source,
  profile,
  learning_time,
  cards,
  minigames,
  npc_summaries,
  local_settings,
  privacy
}
```

- `learning_time`：从 `learning_record.learning_seconds/session_count` 读取；没有时回退到 `game_state` 同名字段，并标记占位。
- `cards`：汇总 `learning_record.card_states` 中已接触、已收集、已玩过和 shiny 卡片。
- `minigames`：汇总 `learning_record.minigame_results` 的次数、时长、奖励 coins、最佳分数和最近结果。
- `npc_summaries`：透传并计数 `learning_record.npc_summary_refs`，由 `ConversationSummaryService` 本地摘要 stub 写入。
- `local_settings`：合并 `profile/game_state/learning_record.local_settings`。
- `privacy`：固定声明 `account_enabled=false`、`network_enabled=false`、`recording_enabled=false`。
