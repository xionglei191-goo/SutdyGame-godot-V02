# Round176 Place Interior Shells

Proof-only cozy town place interior and front-area shell candidates.

These PNGs are building blocks for art-direction review only. They are intentionally not mapped into runtime, ThemeProfile, AssetResolver, data, shared tests, or docs. Broad wall, floor, and front-area pads are documented as `region_band_not_sprite` and must be layered behind actors, furniture, and Round174 place prefabs rather than treated as pickup sprites.

Generated raws use `/home/xionglei/GameProject/tools/image_generator.js`; finals are locally normalized and validated by `normalize_validate_manifest.py` because generator transparency is not trusted unless measured.
